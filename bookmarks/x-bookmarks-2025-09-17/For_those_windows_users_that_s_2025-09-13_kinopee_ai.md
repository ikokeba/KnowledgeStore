## Content

For those windows users that suffer terminal use in Cursor (or vscode), you can actually change default terminal in settings:
-> ctrl + shift + p > "Open VS Code Settings"
-> Search for "Default Profile: Windows"
-> Click button "Edit in settings.json"
-> Add
